Galaga Game

A Java implementation of the classic arcade game Galaga.

View on GitHub Download JAR

How to Play:

  1. Click the "Download JAR" button above to save the file to your computer.
  2. Ensure you have Java Runtime Environment (JRE) installed on your system.
  3. Navigate to the folder where the `.jar` file is saved.
  4. Double-click the `.jar` file to launch the game. Alternatively, run it via the command line: 
    java -jar GalagaGame.jar

Sword-Fish Project

A creative hybrid model of a sword and a fish created using Maya, showcasing advanced modeling and rendering techniques.

Download PDF Download Maya Files

Project Highlights:

  • Modeling: Used polygon modeling, NURBS, and curves for detailed shapes.
  • Lighting: Arnold Physical Sky and Directional Light for natural and focused lighting.
  • Texturing: Applied metallic shaders and UV Mapping for accurate textures.
  • Rendering: High-quality stills rendered using Arnold at 1920x1080 resolution.
